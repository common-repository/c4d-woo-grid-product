== Description ==

- Display product by grid style

- Ajax update when change category

- Support Load More Product

This plugin is used in C4D Vision - Responsive  WooCommerce Theme

You can view here http://coffee4dev.com/responsive-woocommerce-theme-c4d-vision/

== Installation ==

<h3>From your WordPress dashboard</h3>

<ol>
<li>Visit ‘Plugins > Add New’</li>
<li>Search for ‘c4d woo grid product</li>
<li>Activate plugin from your Plugins page.</li>
</ol>
 

<h3>From WordPress.org</h3>

<ol>
<li>Download Woo Countdown Sale Product.</li>
<li>Upload the ‘c4d-woo-grid-product directory to your ‘/wp-content/plugins/’ directory, using your favorite method (ftp, sftp, scp, etc…)</li>
<li>Activate plugin from your Plugins page.</li>
</ol>

<h3>Shortcode</h3>
[c4d_woo_gp category="10, 11, 12" cols="4" loadmore="0" loadmore_text="More Product" count="8" order="date|best_selling_products|best-selling|top-rated"]
